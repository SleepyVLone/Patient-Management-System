# NHS Patient Data System

## Details:

This program was a project assigned to me by my university as part of the Fundamentals of Software Development module.

## How to Run:

Compile using clang++ with the provided sqlite3.o file and run ./nhs

## Features:

- Role-based login system (Patient, Nurse, Doctor, Pharmacist)
- Patient registration and record management
- Treatment cost calculations
- Patient statistics and analytics
- SQLite database storage
